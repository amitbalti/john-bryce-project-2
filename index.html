About the project:

This project is meant to display 100 coins from "https://api.coingecko.com/api/v3/coins/list" API.
The user can choose up to 5 coins, which their live rate data, will be displayed in the Live Reports section.
Also, in the Home page, all the 100 coins will be displayed, and in the 'More Info' button the user can see the coin rates in USD ($), EURO (€), and ILS (₪).
Finally, the user can also search for certain coins in the top search bar.
